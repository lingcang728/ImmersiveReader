# Unified shell reference

`unified-shell-gpt-image-2.png` is the product layout reference approved for
the ImmersiveReader 1.1.0 consolidation.

Use it for information architecture only:

- one application shell with Library, Acquisition, Book Detail, and Reader workspaces;
- a persistent left navigation rail and a contextual task/event panel;
- clear progression from bookshelf to acquisition, detail, and reading;
- a default 1280×800 desktop composition, with the right panel collapsing below 1100px.

Do not mechanically copy the mockup's gold palette. The implemented product
keeps the existing ImmersiveReader themes, blue link/accent language, Focus
Mode behavior, and viewport-anchor rules.
